\chapter{\texttt{simp} と \texttt{rw}}
\label{chap:ch08_simp_rw}

\begin{chapterabstract}
本章では、\texttt{rfl} の次の証明道具として \texttt{simp} と \texttt{rw} を導入する。
\texttt{simp} は、Lean が知っている単純化規則で式を整理する。
\texttt{rw} は、すでに証明済みの等式を使って式を書き換える。
どちらも \texttt{rfl} だけでは届かない場面で使う、等式推論の基本道具である。
\end{chapterabstract}

\begin{learninggoals}
\item \texttt{simp} を使って、分岐や明らかな整理を含む等式を証明できる。
\item \texttt{rw} を使って、既知の等式でゴールを書き換えられる。
\item \texttt{rfl}・\texttt{simp}・\texttt{rw} の役割を読み分けられる。
\end{learninggoals}

\section{この章を読む理由}

これまでの証明は \texttt{rfl} だけで閉じてきた。
しかし、分岐の整理や、別の場所で証明した等式を使いたい場面では、\texttt{rfl} だけでは足りない。
本章では、その二つの場面に対応する \texttt{simp} と \texttt{rw} を導入する。
これらは、次章の等式推論とリファクタリング証明で中心的に使う。

\section{\texttt{simp} で明らかな整理を任せる}

\texttt{simp} は、Lean が知っている単純化規則を使って式を整理する道具である。
たとえば、条件が \texttt{true} に決まっている分岐の整理に使える。

\begin{listing}[htbp]
\begin{minted}{lean4}
def chargeShipping (isMember : Bool) (shipping : Nat) : Nat :=
  if isMember then 0 else shipping

theorem member_shipping_zero (shipping : Nat) :
    chargeShipping true shipping = 0 := by
  simp [chargeShipping]

-- これは任意の shipping については証明できない。
-- shipping が 0 とは限らないため。
-- theorem nonmember_shipping_zero (shipping : Nat) :
--     chargeShipping false shipping = 0 := by
--   simp [chargeShipping]
\end{minted}
\caption{\texttt{simp} で分岐を整理する}
\label{lst:ch08_simp}
\end{listing}

コード\ref{lst:ch08_simp}では、\texttt{chargeShipping true shipping} は \texttt{if true then 0 else shipping} になり、\texttt{simp [chargeShipping]} がこれを \texttt{0} に整理する。
角括弧の \texttt{[chargeShipping]} は「この定義も使って整理してよい」という指定である。
コメントアウトした \texttt{nonmember\_shipping\_zero} は、非会員の場合に \texttt{shipping} がそのまま残るため、任意の \texttt{shipping} については証明できない。

\begin{table}[htbp]
\centering
\scriptsize
\setlength{\tabcolsep}{3pt}
\caption{\texttt{chargeShipping} の分岐の展開}
\label{tab:ch08-simp-branches}
\begin{tabular}{@{}p{0.20\linewidth}p{0.74\linewidth}@{}}
\toprule
式 & 展開後 \\
\midrule
\texttt{chargeShipping true shipping} & \texttt{0} \\
\texttt{chargeShipping false shipping} & \texttt{shipping} \\
\bottomrule
\end{tabular}
\end{table}

\begin{syntaxbox}
\texttt{simp} の記法を固定する。
\begin{itemize}
\item \texttt{simp} は、Lean が知っている単純化規則で式を整理する。
\item \texttt{simp [name]} は、その定義や補題も使って整理する。
\item 何でも証明する魔法ではなく、「計算すれば明らか」な部分を片付ける道具である。
\end{itemize}
\end{syntaxbox}

\section{\texttt{rw} で既知の等式を使う}

\texttt{rw} は rewrite の略で、すでに証明済みの等式を使って式の一部を書き換える。
書き換えた結果、左右が同じ形になれば証明が閉じる。

\begin{listing}[htbp]
\begin{minted}{lean4}
def addTax (price tax : Nat) : Nat :=
  price + tax

def normalizePrice (price : Nat) : Nat :=
  price

theorem normalizePrice_eq (price : Nat) :
    normalizePrice price = price := by
  rfl

theorem addTax_after_normalize (price tax : Nat) :
    addTax (normalizePrice price) tax = addTax price tax := by
  rw [normalizePrice_eq]
\end{minted}
\caption{\texttt{rw} で既知の等式を使って書き換える}
\label{lst:ch08_rw}
\end{listing}

コード\ref{lst:ch08_rw}では、\texttt{normalizePrice\_eq} という等式を使って、\texttt{normalizePrice price} を \texttt{price} に書き換えている。
書き換えの過程は次のように追える。

\begin{table}[htbp]
\centering
\caption{\texttt{rw [normalizePrice\_eq]} の書き換え}
\label{tab:ch08-rw}
\begin{tabular}{ll}
\toprule
段階 & ゴール \\
\midrule
最初のゴール & \path{addTax (normalizePrice price) tax = addTax price tax} \\
\path{normalizePrice price} を書き換え & \path{addTax price tax = addTax price tax} \\
結論 & 左右が同じなので閉じる \\
\bottomrule
\end{tabular}
\end{table}

\begin{leanbox}
\texttt{rfl} は「定義を見れば同じ」、\texttt{simp} は「明らかな整理をまとめて行う」、\texttt{rw} は「既知の等式で置き換える」と読む。
この三つが、次章以降の等式推論とリファクタリング証明の基本語彙になる。
\end{leanbox}

\section{本章のまとめ}

\texttt{simp} は、分岐や明らかな整理を含む等式を片付ける。

\texttt{rw} は、証明済みの等式でゴールを書き換える。

\texttt{rfl}・\texttt{simp}・\texttt{rw} は、それぞれ別の場面で使う等式の道具である。

\begin{summarybox}
この章で新しく出た記法
\begin{itemize}
\item \texttt{simp} \quad 既知の単純化規則で整理する。
\item \texttt{simp [name]} \quad 指定した定義・補題も使って整理する。
\item \texttt{rw [h]} \quad 等式 \texttt{h} でゴールを書き換える。
\end{itemize}
\end{summarybox}
