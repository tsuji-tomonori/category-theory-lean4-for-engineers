# 付録D 画像生成プロンプト

## fig-appD-migration-proof-flow.png

- 対応箇所: 付録D 章冒頭
- 目的: マイグレーション証明で確認する入力条件、変換、逆変換、保存したい性質、CI 境界を一連の流れとして把握できるようにする。
- 画像生成プロンプト:
  日本語の技術書に入れる16:9横長、白背景のフラットなベクター図。上段は「旧データ」→「移行関数」→「新データ」の前向き経路を描き、「新データ」から「旧データ」へ戻る下向きの弧に「戻し関数」と置く。その右に「確認したい性質」を配置し、前向き経路と戻り経路の両方から線をつなぐ。下段は「前提条件 / well-formed」「round-trip」「情報損失の有無」「境界ケース」「バッチ・業務・実装差分」「CI で確認」の6項目を横並びのチェックリストとして配置し、各項目の左に小さなチェック欄を置く。文字列は指定どおり正確に一度だけ描き、英語だけの縦長フローチャート、章番号、図番号、図タイトルは画像内に入れない。余白を広く取り、レビューで走査しやすい構図にする。
- スタイル指定:
  - 日本語技術書向け
  - フラットなベクター図
  - 白背景
  - 余白多め
  - 色数は少なめ
  - 数式は最小限
  - 文字は後から差し替えやすい配置
- 避けること:
  - 写実的な人物
  - 複雑すぎる背景
  - 小さすぎる文字
  - 原典PDFの図に似せすぎること
