# 第7章 画像生成プロンプト

## fig-ch07-pipeline.png

- 対応箇所: 第7章「無名関数・型パラメータ・関数合成」
- 目的: 関数の出力型と次の関数の入力型をつなぐことが関数合成であり、後の圏論における合成の入口になることを示す。
- 画像生成プロンプト:
  日本語の技術書に入れる16:9横長、白背景のフラットなベクター図。左から「入力 x : A」「f : A → B」「中間値 f x : B」「g : B → C」「出力 g (f x) : C」の5ノードを矢印でつなぐ。処理ノード f と g は角丸の箱、値ノードはカードとして見分けられるようにする。中間の型 B を淡い色で強調し、下部に「前の出力型 B が次の入力型 B になる」と置く。文字列は指定どおり正確に一度だけ描き、章番号、図番号、図タイトルは画像内に入れない。数式は最小限にし、余白を広く取る。
- スタイル指定:
  - 日本語技術書向け
  - フラットなベクター図
  - 白背景
  - 余白多め
  - 色数は少なめ
  - 数式は最小限
  - 文字は後から差し替えやすい配置
- 避けること:
  - 写実的な人物
  - 複雑すぎる背景
  - 小さすぎる文字
  - 原典PDFの図に似せすぎること
