## fig-ch22-round-trip-laws.png

- 対応箇所: 第22章「圏論での読み方：同型としてのスキーマ変更」
- 目的: 二つの round-trip law がそろって初めて lossless migration と言えることを示す。
- 画像生成プロンプト:
  左に `UserV1`、右に `UserV2` のノードを配置する。上側に `migrate` の矢印を左から右へ、下側に `rollback` の矢印を右から左へ描く。さらに左側に小さなループとして `rollback ∘ migrate = id`、右側に小さなループとして `migrate ∘ rollback = id` を配置する。二つの法則が同時にチェックされることがわかるように、左右にチェックマークを置く。
